The Long Dark Tea-Time of the Soul

今天好像来不及学oi了，因此就写写我刚刚看完的书吧。

书名就是标题，可以翻译成「灵魂的漫长黑暗的下午茶」（我翻译水平极差），目前好像还没有中文版。这本书的作者是Douglas Adams，也就是「银河系漫游指南」系列的作者。我很小的时候就看过了那个系列，或许根本没看懂（就像当时看的大部分书一样）但仍然很喜欢里面幽默的语言————就算翻译得非常糟糕，也没有完全失去作者的味道。

这本书是Dirk Gently系列的第二本，讲的是一个神奇的私家侦探Drik Gently的故事。他坚称「万物冥冥之中都有联系」，说自己破案的方法是把所有事物连在一起。看了书的标题，一定没有人想到这本书实际上是关于北欧神话的。

当初被人类的祖先们所创造出来的不死的神，也仍然生活在这个世界上，他们就像人类所希望的那样拥有永远的生命，但却早就被人遗忘。就算雷神Thor，一个个子极高的壮汉，拎着他的锤子在大街上走，也没有人会多看他一眼。

看到了结尾才发现，故事暗藏的主线是众神之父奥丁把自己不死的灵魂卖给了一对律师和广告从业者夫妇。他已经非常、非常年迈，而真正想要的其实只是医院里柔软的亚麻布。律师答应为他负担在医院的费用，而他也成为了凡人。

这本书究竟是关于什么的呢？在它滑稽的外表下面，作者是否也有更深一层的意思？我觉得或许它是关于人类和其他生命的相处形式，或许我们不应该把自己放在中间，而是把整个生态圈看成一个整体，每一部分都同样重要。

那些古老的故事，美好或者黑暗，毕竟不应该随着时代的发展而被遗忘。它们中当然也有不少先人的智慧，以及他们生活和梦想的缩影。

‘it’s a hot potato, don’t pick it up, pick it up, pick it up, pass it on, pass it on, pass it on…..’
